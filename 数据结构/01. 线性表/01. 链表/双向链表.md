1. 在单链表中，每个结点只能找到直接后继的结点，而无法找到直接前驱的结点。单链表寻找下一个结点的时间复杂度为 $O(1)$ ， 而寻找上一个结点的时间复杂度为 $O(n)$ 。
2. 双向链表的结点中有两个指针域，一个指向直接后继，另一个指向直接前驱。
3. 双向链表也可以循环。


## 双向链表存储结构实现
```c
typedef struct DuLNode {
    ElemType data;
    struct DuLNode *prior; // 指向直接前驱
    struct DuLNode *next; // 指向直接后继
}DuLNode, *DuLinkList;
```
